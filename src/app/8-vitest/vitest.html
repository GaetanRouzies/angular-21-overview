Vitest - New Angular Test Runner

Angular 21 utilise désormais Vitest comme nouveau framework de test, remplaçant Karma et Jasmine.

Vitest est un framework de test moderne, rapide et optimisé, qui offre une meilleure expérience de développement et des performances supérieures pour les tests unitaires Angular.

Les fichiers de test *.spec.ts utilisent maintenant Vitest avec une API similaire à Jest, mais optimisée pour la vitesse et l'intégration avec Vite.
