Angular MCP Server

Les assistants IA peuvent désormais accéder au serveur MCP (Model Context Protocol) d'Angular.

Le serveur MCP permet aux IA d'interagir directement avec les outils Angular CLI, de rechercher dans la documentation officielle, et d'accéder aux meilleures pratiques et exemples de code Angular.

Cela facilite considérablement le développement Angular avec l'assistance d'IA, car les assistants peuvent utiliser les informations les plus à jour et les plus précises sur Angular.

Configuration MCP

Ce projet contient un fichier .cursor/mcp.json qui configure le serveur MCP Angular CLI pour Cursor.

Ce fichier permet à Cursor de se connecter automatiquement au serveur MCP d'Angular, donnant ainsi accès à toutes les fonctionnalités MCP d'Angular directement depuis l'assistant IA.
